A modern, responsive restaurant website built to provide customers with an engaging online experience. The website showcases the restaurant's menu, signature dishes, services, gallery, and contact information with a clean and user-friendly interface.

## Features

* Responsive design for desktop, tablet, and mobile
* Beautiful hero section
* Interactive food menu
* Featured dishes
* About Us section
* Customer testimonials
* Contact form
* Google Maps integration
* Smooth scrolling navigation
* Fast and optimized performance

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
